# Get-SSHPortForward

## SYNOPSIS
Get a list of forwarded TCP Ports for a SSH Session

## SYNTAX

### Index (Default)
```
Get-SSHPortForward [-SessionId] <Int32>
```

### Session
```
Get-SSHPortForward [-SSHSession] <SshSession>
```

## DESCRIPTION
Get a list of forwarded TCP Ports for a SSH Session

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Get list of configured forwarded ports
```

PS C:\\\> Get-SSHPortForward -Index 0


 BoundHost : 0.0.0.0
 BoundPort : 8081
 Host      : 10.10.10.1
 Port      : 80
 IsStarted : True
